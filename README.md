<p style="text-align:center;">PROOF OF CONCEPT</p>
<p style="text-align:center;">TOY</p>

This is a Self-Hosted MoonBit compiler (written completely in MoonBit) targetting 64-bit Linux. 100% of the source code was created by AI with minimal direction. 

`test_examples.sh` will build the compiler, then use the compiler to compile the examples. then each example's output is compared to the output from the example being run my the official compiler.
